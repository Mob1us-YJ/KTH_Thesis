# 为什么 ACF > STFT > 1D：可视化分析

## 核心发现

通过对比 Waving（挥手）和 Turning（转身）两种典型的**周期性动作**，我们可以清晰地看到三种表示在性能上的差异：

$$\text{Model Accuracy: } ACF > STFT > 1D$$

## 1. 三种表示的核心差异

### 1D CSI Sequence [256, 16]
**特点**：混合时频信息，未经分离
- 行：256 个时间步（~4.6 秒）
- 列：16 个频带
- **问题**：时间和频率维度混在一起，难以分离周期性特征

**从可视化看**：
- 图 (a) 显示红蓝条纹但缺乏清晰结构
- 能量分散，无明显周期性标记

---

### STFT (Spectro-Temporal) [16, 65, 13]
**特点**：将时间和频率分离，但信息分散
- 16 频带
- 65 个频率箱（FFT 输出）
- 13 个时间帧（hop_length=16）

**优势**：
- 展示频域分解
- 可观察到黄色高能区域（集中在特定频率）

**局限**：
- 能量分散在多个频率箱中
- 时间分辨率低（仅13帧 vs 256时间步）
- **难以捕捉周期性周期**（运动的重复频率在低频，但FFT箱数有限）

**从可视化看**：
- 图 (b) 显示黄色条带（集中能量）
- 但相比 1D 没有显著改进时间维度的周期性理解

---

### ACF (Autocorrelation Function) [65, 16]
**特点**：直接捕捉**时间依赖性**和**周期性**
- 65 个延迟值（lag 0 ~ 64）
- 16 个频带
- 每个点 (lag, band) 显示该频带在 lag 延迟处与自身的相关系数

**超群之处**：
1. **清晰的周期性模式**（图 c 的红蓝条纹）
2. **可自动检测周期**（图 e 的绿色峰值标记）
3. **与运动物理对应**：
   - lag=0：自相关最大（总能量）
   - lag=3-5：**周期性峰值**（对应 ~50-100ms 的运动周期）
   - lag>20：相关快速衰减（超过1秒无显著周期性）

**从可视化看**：
- 图 (c) 展示明确的条纹模式（高度周期性）
- 图 (e) 清晰标记周期性峰值：绿色点表示相关性高峰
- **最易被分类器识别**

---

## 2. 为什么 ACF 性能最好？

### 2.1 可分离性 (Separability)

**1D**：混合表示
```
时间-频率混合 → 难以区分类别
Waving vs Turning 的特征可能重叠
```

**STFT**：部分分离
```
时间和频率分离 → 但"何时频率激活"信息丢失
缺少"运动节律"信息
```

**ACF**：完全聚焦于**时间结构**
```
周期性直接可见 → 不同动作的运动节律差异明显
Waving: 挥手的周期性 (~3-5 lag units)
Turning: 转身的周期性 (不同的 lag 分布)
```

### 2.2 从图像看的直观差异

#### Waving（挥手）样本：

**(a) 1D**: 红蓝混乱，无清晰周期
- 不易被 CNN 识别

**(b) STFT**: 黄色条带（高能区）但分散
- 时间帧太少（13帧），周期信息压缩

**(c) ACF**: **清晰的红蓝条纹周期**
- 高相关→低相关→高相关的重复模式
- **完美的周期性指纹**

**(d) 能量**: 时间序列波动
- 显示整体能量动态，但不如周期性结构强

**(e) ACF 周期性**: **绿色峰值标记周期性**
- 自动检测到周期：period ≈ 3.0 lag units
- 清晰的周期指纹特征

## 3. 机器学习的视角

### 为什么 CNN 在 ACF 上表现更好？

**1D 输入 [256, 16]**：
```
Conv kernels 需要同时学习：
- 时间卷积（捕捉动态）
- 频带卷积（捕捉空间关系）
- 学习难度高，特征混合
```

**STFT 输入 [16, 65, 13]**：
```
改进但时间分辨率低：
- 13 时间帧不足以表示细粒度周期性
- 频率分解但计算量大 (16 × 65 × 13 = 13,520 参数)
```

**ACF 输入 [65, 16]**：
```
理想的特征空间：
- Lag 维度直接编码周期性
- Conv kernel 轻松捕捉 "周期指纹"
- 特征紧凑 (65 × 16 = 1,040 参数)
- 不同动作的周期特征分离度最高 ✓
```

## 4. 数学直觉

### ACF 的周期性捕捉能力

对于周期信号，ACF 在周期处产生峰值：

$$\rho(\tau) = \frac{E[x_t \cdot x_{t+\tau}]}{E[x_t^2]}$$

- **Waving（挥手）**：周期 = 3-5 lag units → ACF 在 lag=3,6,9,... 处有峰值
- **Turning（转身）**：周期 = 稍大不同 → ACF 峰值位置不同
- **Sitting（坐着）**：基本无周期 → ACF 快速衰减

这些不同的 ACF 模式是类别之间最易分离的特征！

## 7. 实际应用启示

### 何时选择哪种表示？

| 情景 | 推荐 | 原因 |
|------|------|------|
| **周期性运动**（走路、挥手、转身） | **ACF** | 周期特征是最强判别器 |
| **复杂频谱变化** | STFT | 需要精细的时频分离 |
| **快速推理** | 1D | 最轻量，实时应用 |
| **学术论文** | **ACF** | 性能最好 + 可解释性高 |
